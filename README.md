## Showcase

This program prints the 10th number of Fibonacci sequence:

```
= fibonacci -> index
  ? (> index 1)
    +
      fibonacci (- index 1)
      fibonacci (- index 2)
    index

console.log (fibonacci 10)
```

It prints `55` to the console.
